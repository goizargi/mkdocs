IFTOP

Iftop es una aplicacion similar a top, pero esta centrada especificamente para la monitorizacion de la red. Iftop nos permitira conocer una gran cantidad de detalles respecto a la red, como el ancho de banda consumido, las conexiones TCP/UDP que hemos establecido, direcciones IP y puerto y mucha mas informacion.

Instalacion
apt install iftop
