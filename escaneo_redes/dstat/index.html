DSTAT

Un monitor algo menos conocido que los anteriores, pero que tambien es muy util. Esta herramienta sirve para generar estadisticas de uso del sistema operativo, tanto a nivel de CPU, disco, memoria RAM, como tambien de la red local. De hecho, podremos habilitar o no el estado de la red e incluso elegir las interfaces.

Instalacion

        apt install dstat
