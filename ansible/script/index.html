Autodescubrimiento de equipos

Variables

  • host_count: Cuenta la cantidad de equipos que hay disponibles [ansible/hosts]

  • ip_address: Obtiene la IP del equipo

  • hostname: Obtiene el nombre del equipo

  • so: Obtiene el sistema operativo de cada host

  • dns: Obtiene las direcciones que usa cada host para el DNS

  • Task 1: Crear el fichero para almacenar todos los datos de los hosts

  • Task 2: Agregar el resto de datos al fichero
